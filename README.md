💱 Convertisseur de devises 💱

Description du projet :
Ce projet est une application web simple permettant de convertir un montant d’une devise vers une autre en temps réel.
L'utilisateur peut choisir la devise de départ et la devise cible, saisir un montant, et obtenir instantanément le résultat de la conversion.
L'application utilise une API de taux de change pour récupérer les valeurs actualisées.

Technologies utilisées :
- HTML : structure de la page
- CSS : mise en forme (design rose et moderne)
- JavaScript : logique de conversion et interaction utilisateur
- API ExchangeRate : récupération des taux de change en temps réel
- LocalStorage : sauvegarde du dernier résultat
  
Fonctionnalités principales :
- Conversion de devises en temps réel
- Large choix de devises (EUR, USD, TND, etc.)
- Bouton ⇄ pour inverser les devises
- Affichage dynamique du résultat
- Message d’erreur si montant invalide
- Sauvegarde du dernier résultat (LocalStorage)
- Interface simple, claire et responsive

Nouveautés explorées :
Pendant ce projet, j’ai appris à :
- Utiliser une API externe avec fetch
- Manipuler le DOM en JavaScript
- Gérer les événements (input, change, click)
- Utiliser le LocalStorage pour sauvegarder des données
- Améliorer le design avec CSS moderne (flexbox, gradients)

Difficultés rencontrées :
- Comprendre comment fonctionne une API (requête / réponse JSON)
- Gérer les erreurs (API indisponible, devise non trouvée)
- Mettre à jour automatiquement le résultat
- Styliser correctement les éléments avec CSS

Solutions apportées :
- Consultation de la documentation de l’API
- Utilisation de try...catch pour gérer les erreurs
- Ajout d’événements (addEventListener) pour rendre l'application dynamique
